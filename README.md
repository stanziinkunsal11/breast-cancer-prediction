🧠 Breast Cancer Prediction Web App

This project is a machine learning-based web application that predicts whether a tumor is malignant (cancerous) or benign (non-cancerous) using a trained Logistic Regression model. The app takes 30 medical features as input and provides an instant prediction through an interactive interface built with Gradio.

🚀 Features
Predicts breast cancer diagnosis (Malignant / Benign)
Uses Logistic Regression (Scikit-learn)
Interactive web interface with Gradio
Real-time predictions
Deployed on Hugging Face Spaces

📊 Dataset
Uses the Breast Cancer Wisconsin Dataset
Contains 30 numerical features computed from digitized images of a breast mass

🛠️ Tech Stack
Python
NumPy
Scikit-learn
Joblib
Gradio
